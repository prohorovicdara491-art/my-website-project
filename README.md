**Проект по учебной практике (ПМ.02)**  
**Специальность:** 09.02.07 Информационные системы и программирование  
**Тема сайта:** Ветеринарная клиника — услуги и запись

---

## 📌 О проекте

**«Доктор Хвост»** — это сайт ветеринарной клиники, где владельцы животных могут:

- 🩺 узнать об услугах и ценах
- 👨‍⚕️ познакомиться с врачами
- 📚 почитать полезные статьи о здоровье питомцев
- 📞 записаться на приём через форму обратной связи
- 💬 общаться на форуме с другими владельцами

---

## 🚀 Технологии

| Технология | Назначение |
|------------|------------|
| LAMP (Linux, Apache, MySQL, PHP) | Серверная платформа |
| WordPress | Система управления контентом |
| GitHub | Хранение кода и документации |
| Contact Form 7 | Форма обратной связи |
| bbPress | Форум для общения владельцев |

## 📢 Форум

На сайте установлен форум **bbPress** для общения владельцев домашних животных.

### 📊 Статистика форума

| Показатель | Значение |
|------------|----------|
| Форумов | 1 |
| Категорий | 3 |
| Тем | 6+ |
| Сообщений | 6+ |

### 🏷️ Категории форума

- 🩺 **Здоровье и болезни** — вопросы о лечении, симптомах и профилактике
- 🐾 **Уход и содержание** — питание, гигиена, воспитание и комфорт питомцев
- 🚑 **Первая помощь** — экстренные ситуации до визита к врачу
